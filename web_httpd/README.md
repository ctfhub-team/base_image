# web_httpd

## 环境版本

- Debian 12 (bookworm) amd64
- Apache/2.4.57

默认软件源为 http://mirrors.ustc.edu.cn/debian

## 环境变量

- FLAG: 设置flag，如不覆盖flag.sh则默认写入到`/flag`

## 配置文件

- apache2.conf: /etc/apache2/apache2.conf
- vhost: /etc/apache2/sites-available/000-default.conf

请查看`example`文件夹
