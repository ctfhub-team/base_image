# web_httpd_php_7.4

## 环境版本

- Debian 11 (bullseye) amd64
- Apache/2.4.51
- PHP 7.4.27

默认软件源为 http://mirrors.tencent.com/debian

## 环境变量

- FLAG: 设置flag，如不覆盖flag.sh则默认写入到`/flag`

## 配置文件

- php.ini: /usr/local/etc/php/php.ini
- 000-default.conf: /etc/apache2/sites-available/000-default.conf
