# web_nginx_php_5.6

## 环境版本

- Debian 9 (stretch) amd64
- nginx/1.10.3
- PHP 5.6.40

默认软件源为 http://mirrors.tencent.com/debian-archive/debian

## 环境变量

- FLAG: 设置flag，如不覆盖flag.sh则默认写入到`/flag`

## 配置文件

- php.ini: /usr/local/etc/php/php.ini
- nginx.conf: /etc/nginx/nginx.conf
