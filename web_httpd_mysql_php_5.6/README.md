# web_httpd_mysql_php_5.6

## 环境版本

- Debian 9 (stretch) amd64
- Apache/2.4.25
- PHP 5.6.40
- MariaDB 10.1.45-MariaDB-0+deb9u1

已安装mysql mysqli pdo_mysql三个扩展

## 环境变量

- FLAG: 设置flag，如不覆盖flag.sh则默认写入到`/flag`，如需要单独设置flag存储到数据，请查看示例

## 配置文件

- php.ini: /usr/local/etc/php/php.ini
